
BOT DE SERVIÇOS - WORLD OF WARCRAFT TBC

FUNCIONALIDADES
- cadastro de profissões
- busca de crafters
- criação de pedidos
- lista de pedidos

PASSO A PASSO

1) Instale Python
https://python.org

2) Extraia os arquivos

3) Renomeie
.env.example -> .env

4) Coloque seu token do Discord dentro do .env

DISCORD_TOKEN=SEU_TOKEN

5) Abra PowerShell na pasta do projeto

6) Instale dependências

pip install -r requirements.txt

7) Rode o bot

python main.py

HOSPEDAGEM GRATUITA

Render.com

1) Crie conta
2) Crie novo Web Service
3) Suba os arquivos
4) Render usará render.yaml automaticamente
